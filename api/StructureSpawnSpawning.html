StructureSpawn.Spawning

当前正在孵化的 creep 的详细信息，可以通过StructureSpawn.spawning属性进行访问。

directionsarray

一个指示了出生方向的数组，参见 StructureSpawn.Spawning.setDirections.

namestring

新 creep 的名字。

needTimenumber

完成孵化总共需要的时间。

remainingTimenumber

剩下的时间。

spawnStructureSpawn

一个到对应 spawn 的链接。

cancel() 

Game.spawns['Spawn1'].spawning.cancel();

立即取消孵化。不返还消耗的资源。

Return value

如下错误码之一：
constantvaluedescription
OK0

这个操作已经成功纳入计划。

ERR_NOT_OWNER-1

你不是该 spawn 的所有者。

setDirections(directions) 

Game.spawns['Spawn1'].spawning.setDirections([RIGHT, TOP_RIGHT]);

设置出生方向，以使它们在生成时应移动到的位置。

parametertypedescription
directionsarray<number>

包含如下方向常量的数组:

  • TOP
  • TOP_RIGHT
  • RIGHT
  • BOTTOM_RIGHT
  • BOTTOM
  • BOTTOM_LEFT
  • LEFT
  • TOP_LEFT

Return value

如下错误码之一：
constantvaluedescription
OK0

这个操作已经成功纳入计划。

ERR_NOT_OWNER-1

你不是该 spawn 的所有者。

ERR_INVALID_ARGS-10

无效的方向数组
