Game.shard

描述当前执行脚本的 shard 对象。

Game.shard.namestring

shard 的名称。

Game.shard.typestring

目前总是等于 normal。

Game.shard.ptrboolean

这个 shard 是否属于 PTR.

Game.shard.accessboolean

您当前是否有访问此 shard 的权限。 在非限制性 shard 上总是 true。在限制性 shards 上, 需要激活 ACCESS_KEY 资源或激活无限访问权限。 使用 Game.shard.activateAccess 激活访问。

Game.shard.accessTimenumber

这个 UNIX毫秒时间戳 为激活访问权限的时间。 无访问限制的 shard 或未激活访问权限时此属性未被定义。

Game.shard.activateAccess() 

if(Game.shard.access && Game.shard.accessTime && ((Game.shard.accessTime - Date.now()) < 1000*60*60*24*7)) {
    Game.shard.activateAccess();
}

激活对当前受限 shard 30天的访问授权。 此方法将消耗一个 ACCESS_KEY 资源（详见 Game.resources）。 这个方法仅适用于受限的 shard (定义了 Game.shard.access 的 shard)。

返回值

如下错误码之一:
constantvaluedescription
OK0

这个操作已经成功纳入计划。

你账户中的 accessKey 资源不足。

ERR_INVALID_TARGET-7

这个 shard 不受限。

ERR_FULL-8

你的访问权限是无限的。
