StructureLink

将能量远程传输到同一房间的另一个 Link 中。

控制器等级
1-4 —
5 2 links
6 3 links

transferEnergy(target, [amount]) 

const linkFrom = Game.rooms['W1N1'].lookForAt('structure', 10, 25)[0];

const linkTo = linkFrom.pos.findInRange(FIND_MY_STRUCTURES, 2,
    {filter: {structureType: STRUCTURE_LINK}})[0];

linkFrom.transferEnergy(linkTo);

将能量远程传输到同一房间中任何位置的另一个 Link 中。

parametertypedescription
targetStructureLink

目标对象。
amount (可选)number

将要传输的能量值。如果省略，所有能量都将被传输。

返回值

如下错误码之一：
constantvaluedescription
OK0

这个操作已经成功纳入计划。

ERR_NOT_OWNER-1

你不是该 link 的所有者。

ERR_NOT_ENOUGH_RESOURCES-6

这个建筑内的资源少于给定的数量。

ERR_INVALID_TARGET-7

目标不是一个有效的 StructureLink 对象。

ERR_FULL-8

目标无法接受更多能量。

ERR_NOT_IN_RANGE-9

目标太远了。

ERR_INVALID_ARGS-10

资源数量不正确。

ERR_TIRED-11

该 link 仍在冷却中。

ERR_RCL_NOT_ENOUGH-14

房间控制器等级不足以使用该 link。
